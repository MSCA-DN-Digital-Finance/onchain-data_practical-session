# Onchain Data: A Practical Session

## Getting Started 

- clone this repo : 

```
git clone https://github.com/MSCA-DN-Digital-Finance/onchain-data_practical-session.git
```

- Get an API key for TheGraph 

    - make an account on: https://thegraph.com/subgraphs/

- Add API to .env 

```
GRAPH_API_KEY=your_key_here
```
- Run Jupyter Notebook
